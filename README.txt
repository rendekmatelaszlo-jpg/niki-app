Niki 4.0 PWA

Új elemek a 4.0-ban:
- új jelenet/mód rendszer: normál, fókusz, közeli, játék, nyugodt
- beszélgetési sorozat (streak)
- fókusz cél mentése
- mentés importálása JSON fájlból
- fejlettebb emlékezet minták
- gyors fókusz és kedves üzenet gombok

Feltöltés:
1. a teljes mappa tartalmát töltsd fel a GitHub repository gyökerébe
2. Settings > Pages > Deploy from branch > main
3. Chrome-ban megnyitva telepíthető

Megjegyzés:
Ez továbbra is helyi, szabályalapú PWA. Nincs külső AI szerver.
