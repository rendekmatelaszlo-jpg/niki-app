NIKI 7.0 Android projekt

Ez egy Android Studio projektforrás a NIKI 7.0 prototípushoz.

Mit tud jelenleg:
- nőies, futurisztikus lila dizájn
- AI avatar aura effekttel
- hangulatváltás: Nyugodt / Pajzán / Komoly
- akkumulátorkímélő mód kapcsoló
- chat felület
- "NIKI gondolkodik" állapot
- beszéd animáció és hanghullám effekt

APK készítés Android Studio-ban:
1. Csomagold ki a ZIP-et
2. Nyisd meg a projektet Android Studio-ban
3. Várd meg, amíg a Gradle szinkron lefut
4. Menü: Build -> Generate App Bundles or APK -> Build APK

Package név:
com.mate.niki

Verzió:
7.0
